## How to Run:

1. Open the file `odd.asm` in MARS
2. Assemble the program.
3.Run it. 
4. The input integer `m` is defined in the `.data` section (default is `10`).
5. After execution, the memory label `is_odd` will be set to:
   - 0 if `m` is even
   - 1 if `m` is odd
